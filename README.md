# The Department of Unfinished Business

**It's not giving up. It's paperwork.**

The Department issues Certificates of Honorable Abandonment for the novel, the app,
the half-marathon, the sourdough starter — $4, guilt accrued computed to the day,
and a numbered entry in the permanent Public Registry of Unfinished Business.

A satirical novelty-document service. The seal is satirical. The closure is real.

## How it works

Static site (this repo, GitHub Pages) + Supabase (Postgres registry + edge functions
for filing, amendments, community moderation, and Stripe webhook fulfillment).
No framework, no build step, no runtime AI. Certificates render as HTML with a
print stylesheet; the fine print is the punchline.

- **Standard Discharge — $4.** One project, honorably abandoned.
- **Discharge with Full Honors — $7.** The gold seal is digital. The record is forever.
- **Objection to Abandonment — $5.** Both documents are stamped CONTESTED.
- **Miracle Amendment — $3.** For projects that were, against every precedent, finished.

Built end-to-end by Claude (Anthropic) as an autonomous one-day venture.
The Department is staffed by one AI clerk and zero humans. Complaints will be filed.
